# Thomas J. Watson Digital Museum

A modern educational project inspired by the **IBM Skills Network --
Creating a Simple Web Page** lab.

Instead of reproducing the original exercise, this project expands it
into a small responsive website that demonstrates HTML5, CSS3 and
JavaScript in a practical way.

------------------------------------------------------------------------

# Objectives

Build a complete web page while practicing:

-   HTML5 semantic elements
-   CSS layout and styling
-   Responsive design
-   Forms
-   Tables
-   Lists
-   Images
-   Hyperlinks
-   JavaScript interaction
