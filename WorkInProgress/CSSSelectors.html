Understanding CSS Selectors

Tags • IDs • Classes

What are CSS Selectors?

CSS Selectors identify HTML elements and apply styles to them.

  • Tag Selector → Styles every element of the same HTML tag.
  • ID Selector → Styles one unique element.
  • Class Selector → Styles one or many elements that share the same class.

1. Tag Selectors

This paragraph is styled automatically because every <p> element receives the same CSS.

h1{ color:#0d6efd; } p{ color:#444; }

2. ID Selectors

This box uses the selector #important because its id is unique.
#important{ background:#fff3cd; padding:15px; }

3. Class Selectors

Classes can be reused many times.

✔ HTML is the structure.

✔ CSS styles the page.

⚠ JavaScript adds behavior.

This word is highlighted using a CSS class.

.highlight{ background:gold; }

Summary

  • Tag selectors style all elements of one type.
  • ID selectors target one unique element.
  • Class selectors allow reusable styles.
  • Classes are the most commonly used selectors in modern websites.
  • Combining selectors makes CSS powerful and maintainable.
