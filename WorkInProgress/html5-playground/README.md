# HTML5 Playground

Interactive educational project demonstrating the most common HTML elements.

## Covered Topics

- HTML document structure
- DOCTYPE
- Headings (H1-H6)
- Paragraphs
- Line breaks
- Hyperlinks
- Ordered and unordered lists
- Tables
- Images
- Basic CSS styling
- External JavaScript
