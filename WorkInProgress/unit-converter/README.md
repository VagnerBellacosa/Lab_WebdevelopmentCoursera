HTML5 Unit Converter Lab
Overview
This project is a complete educational example based on the IBM Skills
Network -- HTML5 Unit Conversion Lab.
The application demonstrates how to build a semantic HTML5 web page,
organize forms with structural elements, apply responsive CSS, and use
JavaScript to perform real-time unit conversions.
---
Features
Celsius → Fahrenheit conversion
Kilograms → Pounds conversion
Kilometers → Miles conversion
Responsive layout using Flexbox
Semantic HTML5 structure
Internal page navigation
Organized forms using Fieldset and Legend
Simple and readable JavaScript
