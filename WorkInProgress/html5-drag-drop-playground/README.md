# HTML5 Drag & Drop Playground

Interactive project demonstrating the HTML5 Drag and Drop API.

## Features

- Basic drag and drop example
- Event reference table
- Common use cases
- External CSS and JavaScript
- GitHub-ready structure

## Project Structure

```text
html5-drag-drop-playground/
├── index.html
├── style.css
├── script.js
├── README.md
├── LICENSE
├── .gitignore
├── assets/
├── docs/
└── examples/
```

## API Events

- dragstart
- dragover
- drop
- dragend

## Run

Open `index.html` in a browser or use VS Code Live Server.
